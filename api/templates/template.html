WaterUP

We help farmers make better irrigation decision

Under the backdrop of climate change, orchards around the world are facing unprecedented challenges of water scarcity. The WaterUP API is created for growers and orchard managers to inform their irrigation descision with Machine Learning to predict water stress in plants

Using the waterUP api is simple. To query the water potential forecast in up to 3 days in advance, simple go to the endpoint: /forecast/period={period}, where {period} is an integer of number of hours you wish to forecast from now rounding up to the next hour
